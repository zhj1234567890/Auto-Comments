# Auto-Comments
A tool to help teachers write comments.

一个工具，帮助教师撰写评语。

# Progress
The underlying logic and some key parts have been completed. The first version is expected in August 2023.

目前已经完成底层逻辑和一些关键部分。预计在2023年8月发行第一个版本。
